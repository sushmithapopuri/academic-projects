# FUTURE SCOPE

Creating one single application that performs these functionalities in a robust manner:

1. Collect stock data for all stocks and news from web and save it for analysis
2. Background cleaning of data
3. Better models to predict market sentiments
4. Performing continuous analysis on data to understand and redefine intraday trends
5. In time communication of the trend details to end users
6. Automation of trading in a safe fashion
